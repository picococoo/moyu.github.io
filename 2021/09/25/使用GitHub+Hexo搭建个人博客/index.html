使用GitHub+Hexo搭建个人博客

使用前需要电脑安装好Git+node.js

安装 node.js

brew install node

使用 npm 安装 hexo

sudo npm install -g hexo-cli

安装完成后，在Desktop创建一个blog文件夹，在该文件夹下初始化博客
cd ~/Desktop && mkdir blog && cd blog

在该文件件目录下执行初始化操作
hexo init

将本地博客关联到Github

github新建仓库：moyu.github.io
然后开始修改配置文件 ~/Desktop/blog/_config.yml

push本地博客到github

hexo g #生成
hexo d #部署

常用命令

hexo p => hexo publish
hexo g => hexo generate #生成
hexo s => hexo server #启动服务预览
hexo d => hexo deploy #部署
